### 1、多模块构建

**mvn install**

初始化需要进行下载plugins相关的信息，后续可以过滤相关的下载以及测试代码

**mvn -o install** (将会离线打包)

**mvn -DskipTests install** (打包不执行测试用例)

**mvn clean package -pb** (打包使用不用环境下面的配置文件)
